Get Started in Seconds

Install VraithUI and start building beautiful UIs

1

Install the package

npm install -D vraithui tailwindcss
2

Add to your Tailwind config

// tailwind.config.js
const vraithui = require('vraithui').default

module.exports = {
  plugins: [vraithui]
}
3

Optional: Import runtime helpers

import VraithUI from 'vraithui/runtime'
✓

Start building!

You're all set. Use any of the 51 components in your project.
