How to Use Themes

1. Set Theme via HTML Attribute

Add the data-theme attribute to your HTML element:

<html data-theme="cyberpunk">
  <!-- Your content -->
</html>

2. Switch Themes with JavaScript

Use the VraithUI runtime to change themes dynamically:

import { VraithUI } from 'vraithui/runtime';

// Set a specific theme
VraithUI.theme.set('dark');

// Toggle between themes
VraithUI.theme.toggle(['default', 'dark']);

// Get current theme
const currentTheme = VraithUI.theme.get();

3. Create a Theme Switcher

Build a dropdown or button group for theme selection:

<select class="select" onchange="VraithUI.theme.set(this.value)">
  <option value="default">Default</option>
  <option value="dark">Dark</option>
  <option value="cyberpunk">Cyberpunk</option>
  <!-- More themes... -->
</select>
